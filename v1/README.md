* 用戶登入/註冊

* 健康記錄與提醒
* 圖片上傳（如寵物照片）
* 地圖與醫院推薦

- my pet
    - 三個區塊
        - 健康記錄
            - 添加寵物健康檔案，記錄疫苗接種、體重變化、醫療紀錄等。
            - 飲食指導
                提供針對不同品種、年齡和健康狀況的飲食建議。計算每日建議的熱量和營養需求。
        - 行為訓練
            - 提供基本的訓練指南，如坐下、握手、如廁訓練。記錄訓練進度。
        - 提醒功能
            - 提醒主人給予疫苗、驅蟲、洗澡或剪指甲。
            - 緊急指南常見緊急情況的應對指南，如中毒、外傷處理。醫院與商店地圖推薦附近的動物醫院或寵物店。
        
- social
    - 提供交流平台，讓飼主分享心得和提問。可以發文，可以分類不同品種推薦適合的飼料、玩具和其他用品。
